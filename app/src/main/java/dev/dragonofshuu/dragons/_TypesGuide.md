Dragon Types
=

There are many types of dragons. The main type names are 
def no joke stolen from Genshin impact. Anyway, the types 
and their correspondence are listed below.

 - __Anemo__
   - Wind
 - __Cryo__
   - Ice
 - __Dendro__
   - Nature
 - __Electro__
   - Electric
 - __Geo__
   - Earth
 - __Hydro__
   - Water
 - __Pyro__
   - Fire
